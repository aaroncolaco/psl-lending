# psl-lending
Service API to lend/rent resources using Ethereum to log events

## Config

* Create files `notifications/ABI.txt` & `notifications/contractAddress.txt`
```console
touch notifications/ABI.txt
touch notifications/contractAddress.txt
```
Paste the ABI and the contract address in them

* Update `config/config.json` and `config/firebase-admin.json` if needed
